# Real-Time Emotion Detection System

This project is a real-time Emotion Detection system that analyzes human facial expressions using a camera and predicts emotions using deep learning models.

The system captures live video input, detects faces, and identifies emotions such as happy, sad, angry, surprise, fear, and neutral in real time.

## 🚀 Features
- Real-time face detection using Dlib
- Emotion recognition using DeepFace
- Live webcam emotion analysis
- Multiple emotion classification
- Bounding box visualization on detected faces
- Lightweight and easy to run

## 🧠 Technologies Used
- Python
- OpenCV
- Dlib
- DeepFace
- NumPy

## ⚙️ How It Works
1. Webcam captures live video.
2. Dlib detects human faces.
3. Detected face is processed.
4. DeepFace predicts emotion.
5. Emotion label is displayed on screen.
